# Hi, I'm Sander 👋

**Creative developer at [KNMI](https://www.knmi.nl)** — blending code, hardware, and experiments to build things that shouldn't exist but do.

I work at the intersection of web technology and physical making: browser games you control with your hands, gesture-driven maps, 3D printed objects with embedded logic, and interactive WebGL experiments.

🌍 **Portfolio:** [sanderdesnaijer.com](https://www.sanderdesnaijer.com)  
📍 Maasdam, Netherlands
